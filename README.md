# API Rest -  Validar CPF
Projeto em Spring que contém uma API para validação do CPF.

### 1) Tecnologias usadas
- Spring Web;
- Swagger;
- Docker;

### 2) API Rest

Endereço: http://host:port/users/{cpf}
